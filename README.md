## How to run

- Install [godotenv](https://github.com/joho/godotenv) (optional)
- Install [air]() (optional)
- copy `.env.example` to `.env` and fill in the values
```console
cp .env.example .env
```

- run server

#### with godotenv
```console
godotenv -f .env go run cmd/server/main.go
```

#### with air
```console
air
```
